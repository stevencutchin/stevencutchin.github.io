How Pixels Are Stored in Memory

üîë Memory Layout: Images store pixels in a continuous array where all 4 channels of one pixel are grouped together. This is called "interleaved" format and is the standard for WebGL, Canvas, and most graphics APIs.

üí° When to Use Each Format:
  • 8-bit: Default choice for most applications. Compatible everywhere, small file sizes.
  • 16-bit integer: When you need smooth gradients without banding, especially for editing/processing.
  • 16-bit float (Half): HDR rendering where you need brightness > 1.0. Good balance of precision and memory.
  • 32-bit float: Maximum precision for intermediate calculations, compositing multiple layers, or scientific data.
  • RGB10A2: Modern HDR displays. More colors than 8-bit but same memory footprint.
