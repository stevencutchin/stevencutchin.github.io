Creating a Google Maps View App

In this app we will walk through the steps needed to create your own simple Google Map App.

Google Maps is a sophisticated platform tools built by Google that lets you add Map functionality to your own Android App.

It has a host of features that are available for your use. You can find out detailed information at: https://developers.google.com/maps.

In this walkthrough we will:

  • first create a Google Maps App
  • Modify the App to share the UI with a second component.
  1. Create a New Android Studio Project from MapsActivity Template
    In Android Studio just create a new project using the Maps Activity template. This will create the core activity App and the needed code to build a simple Map App.
  2. Getting the API Code
    Google Maps won't work without an API code. The link to request an API code is in the file 'google_maps_api.xml' in the values folder. After you get the API code you will be able to turn on the Maps object. Now open build.gradle and add compile 'com.google.android.gms:play-services:8.4.0' in dependencies.
