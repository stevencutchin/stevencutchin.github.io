Cookbook for Expo CLI

Install the Expo CLI onto your machine.

  • Install Node.js – this gives you npx, and expo tools.
  • https://nodejs.org/en

Install Expo:

  • npm install expo | yarn add expo

Link to your Expo account.

  • npx expo register
  • npx expo login
  • npx expo whoami

Create a new App Project

  • npx create-expo-app name-of-app

Develop and test on your machine:

  • npx expo start

Build through EAS services for iOS or Android.

  • npm install -g eas-cli
  • eas login or npx expo login
  • eas init – creates a project in your account.

Deploy to The Store of your choice.
