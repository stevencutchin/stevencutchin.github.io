Let's Use Fragments for our Map App

We will modify our map app to use Fragments instead of activities. That way we can adjust our UI display components based on phone orientation:

  1. First we create a Fragment Class to hold our Recycler View
    1. Put RecyclerView into fragment XML file
    2. Put FragmentList into Activity XML file
    3. Modify Activity Code to launch ListFragment instead of RecycleView
  2. Second we replace our Recycler View code in the Activity.xml layout file
  3. Third change our Click Handler to switch visible fragments - end of Intent
  4. Adjust the UI Display based on screen configuration
