Location Sensors: GPS & Magnetometer

★ TRICORDER

Determines position on a planet's surface. Provides directional orientation for navigation to targets.

★ YOUR PHONE

GPS/GNSS receiver triangulates position using satellite signals. Magnetometer acts as a digital compass. Together with WiFi/cell positioning, accuracy reaches 1–3 meters.

1. GPS / GNSS

1.1 What It Measures

The Global Positioning System (GPS) — and more broadly, Global Navigation Satellite Systems (GNSS) — determines the phone's latitude, longitude, and altitude by receiving signals from orbiting satellites. Modern phones can receive signals from GPS (USA), GLONASS (Russia), Galileo (EU), and BeiDou (China) simultaneously, improving accuracy and fix speed.

1.2 How It Works: Trilateration

Each GPS satellite broadcasts a precisely timed signal. The receiver in your phone measures how long each signal took to arrive. Since the signals travel at the speed of light, the travel time gives the distance to each satellite. With distances to at least 4 satellites, the receiver solves a system of equations to find its 3D position (latitude, longitude, altitude) plus a clock correction.

Figure 3.1 — GPS trilateration. Each satellite defines a sphere of possible positions. The intersection of three or more spheres gives a unique fix.

1.3 Assisted GPS (A-GPS)

A cold GPS fix — finding satellites from scratch — can take 30+ seconds. A-GPS solves this by downloading satellite orbital data (almanac and ephemeris) from the internet via WiFi or cellular, so the receiver already knows where to look in the sky. This reduces fix time to 1–3 seconds. Additionally, WiFi and cell tower positioning provides a rough location instantly, which is refined as GPS locks on.

Battery Warning: GPS is one of the most power-hungry sensors on the phone. Continuous GPS polling can drain 25–40% of battery per hour. Navigation apps like Waze manage this by adjusting polling frequency based on context — high frequency while driving, reduced when stationary, off when backgrounded. As a developer, always request the minimum accuracy your use case requires.

2. The Magnetometer

2.1 What It Measures

The magnetometer measures the strength and direction of the magnetic field around the phone. Its primary use is as a digital compass — by detecting Earth's magnetic field, it determines which direction is magnetic north. This is what powers the compass heading in map applications and the blue direction cone on Google Maps.

2.2 How It Works: Hall Effect

Most phone magnetometers use a Hall-effect sensor. When a current flows through a thin conductor in the presence of a magnetic field, the field deflects the charge carriers to one side, creating a measurable voltage difference perpendicular to both the current and the field. Three-axis Hall sensors measure the magnetic field vector in all three dimensions.

Figure 3.2 — Hall effect principle. A magnetic field perpendicular to current flow produces a voltage that is proportional to field strength.
Calibration: You have probably seen your phone’s compass app ask you to wave the phone in a figure-8 pattern. This is magnetometer calibration — the phone needs to measure the local magnetic field from different orientations to compensate for nearby magnetic interference (called “hard iron” and “soft iron” distortion) from the phone’s own components.

3. As a Developer: Expo Location API

import * as Location from 'expo-location';

// Request permission first
const { status } = await Location.requestForegroundPermissionsAsync();
if (status !== 'granted') return;

// Get current position
const location = await Location.getCurrentPositionAsync({
  accuracy: Location.Accuracy.Balanced, // not highest!
});
console.log(location.coords.latitude, location.coords.longitude);

// For compass heading:
import { Magnetometer } from 'expo-sensors';
Magnetometer.addListener(({ x, y, z }) => {
  const heading = Math.atan2(y, x) * (180 / Math.PI);
});
