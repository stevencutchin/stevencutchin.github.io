Pipeline Overview

The Privacy Inspection Pipeline is a fully automated system that transforms raw website URLs into comprehensive privacy investigation documentation. The system integrates blacklight-collector for privacy scanning with custom Python parsers and shell scripts to create end-to-end automation.

Key Features: Automated tracking discovery, structured report generation, evidence documentation, and compliance assessment against California privacy laws (CIPA, CDAFA, Wiretap Act).

üîÑ Automated Pipeline Process

1
URL Input & Sanitization
Shell script accepts URL, sanitizes path for filename safety, handles multiple URL formats
2
Blacklight Privacy Scan
Automated blacklight-collector execution, generates HTML report with tracking analysis
3
Report Parsing & Analysis
Python parser extracts structured data from Blacklight HTML output
4
Investigation Template Generation
Automated markdown report creation with pre-filled findings and testing instructions
5
Evidence Organization
Structured storage under data/ directory with CSV data, HTML reports, and markdown templates

‚ö†Ô∏è Important Distinction

Technical Assessment vs. Legal Analysis: This pipeline performs technical investigation of privacy practices. It documents WHAT is happening (data collection, tracking methods, consent mechanisms). Legal determination of whether these practices VIOLATE specific statutes requires legal counsel and is outside the scope of technical assessment.
